# Notion

## Fiche Technique

## Devento

Bien sûr, voici un tableau récapitulatif des raccourcis clavier de Notion pour Windows, classés par catégorie :

**Raccourcis les plus populaires**

| Raccourci            | Description                                                                     |
|----------------------|---------------------------------------------------------------------------------|
| Ctrl + N             | Créer une nouvelle page (applications de bureau uniquement)                     |
| Ctrl + Maj + N       | Ouvrir une nouvelle fenêtre Notion (applications de bureau uniquement)          |
| Ctrl + clic          | Ouvrir un lien dans un nouvel onglet Notion (applications de bureau uniquement) |
| Ctrl + T             | Créer un nouvel onglet Notion (applications de bureau uniquement)               |
| Ctrl + P ou Ctrl + K | Ouvrir la recherche ou accéder à une page récemment consultée                   |
| Ctrl + L             | Copier l'URL de la page                                                         |
| Ctrl + [             | Revenir à la page précédente                                                    |
| Ctrl + ]             | Aller à la page suivante                                                        |
| Ctrl + Maj + L       | Basculer entre le mode sombre et le mode clair                                  |


**Création et mise en forme de contenu**

| Raccourci             | Description                                                       |
|-----------------------|-------------------------------------------------------------------|
| Entrée                | Insérer une ligne de texte                                        |
| Maj + Entrée          | Créer un saut de ligne dans un bloc de texte                      |
| Ctrl + Maj + M        | Ajouter un commentaire                                            |
| ---                   | Créer une ligne de séparation                                     |
| Ctrl + B              | Mettre le texte sélectionné en gras                               |
| Ctrl + I              | Mettre le texte sélectionné en italique                           |
| Ctrl + U              | Souligner le texte sélectionné                                    |
| Ctrl + Maj + S        | Barrer le texte sélectionné                                       |
| Ctrl + K              | Ajouter un lien hypertexte                                        |
| Ctrl + E              | Appliquer le style de code en ligne au texte sélectionné          |
| Tab                   | Indenter et imbriquer le contenu                                  |
| Maj + Tab             | Désindenter le contenu                                            |
| /turn                 | Transformer un bloc en un autre type de bloc                      |
| /color                | Changer la couleur ou la couleur de surlignage d'un bloc de texte |
| Ctrl + Maj + 0 à 9    | Créer différents types de blocs (texte, titres, listes, etc.)     |
| Ctrl + +              | Zoomer                                                            |
| Ctrl + -              | Dézoomer                                                          |
| Ctrl + Maj + U        | Remonter d'un niveau dans la hiérarchie des pages                 |
| Alt + glisser-déposer | Dupliquer un contenu en le faisant glisser                        |

**Édition et déplacement de blocs**

| Raccourci                            | Description                                                            |
|--------------------------------------|------------------------------------------------------------------------|
| Échap                                | Sélectionner le bloc actuel ou désélectionner les blocs                |
| Ctrl + A                             | Sélectionner le bloc où se trouve le curseur                           |
| Espace                               | Ouvrir une image sélectionnée en plein écran ou quitter le plein écran |
| Flèches directionnelles              | Sélectionner un autre bloc                                             |
| Maj + flèche haut/bas                | Étendre la sélection vers le haut ou le bas                            |
| Alt + Maj + clic                     | Sélectionner ou désélectionner un bloc entier                          |
| Maj + clic                           | Sélectionner un autre bloc et tous les blocs intermédiaires            |
| Retour arrière ou Suppr              | Supprimer les blocs sélectionnés                                       |
| Ctrl + D                             | Dupliquer les blocs sélectionnés                                       |
| Entrée                               | Modifier le texte d'un bloc sélectionné ou ouvrir une page imbriquée   |
| Ctrl + /                             | Modifier ou changer un ou plusieurs blocs sélectionnés                 |
| Ctrl + Maj + flèches directionnelles | Déplacer un bloc sélectionné                                           |
| Ctrl + Alt + T                       | Développer ou réduire toutes les listes déroulantes                    |
| Ctrl + Maj + H                       | Appliquer la dernière couleur de texte ou de surlignage utilisée       |
| Ctrl + Entrée                        | Modifier le bloc actuel (ouvrir une page, cocher une case, etc.)       |


**Styles Markdown**

| Raccourci                        | Description                                      |
|----------------------------------|--------------------------------------------------|
| **texte**                        | Mettre le texte en gras                          |
| *texte*                          | Mettre le texte en italique                      |
| `texte`                          | Créer du code en ligne                           |
| ~~texte~~                        | Barrer le texte                                  |
| * ou - ou + suivi d'un espace    | Créer une liste à puces                          |
| [] suivi d'un espace             | Créer une case à cocher pour une liste de tâches |
| 1. ou a. ou i. suivi d'un espace | Créer une liste numérotée                        |
| # suivi d'un espace              | Créer un titre de niveau 1                       |
| ## suivi d'un espace             | Créer un titre de niveau 2                       |
| ### suivi d'un espace            | Créer un titre de niveau 3                       |
| > suivi d'un espace              | Créer une liste déroulante                       |
| " suivi d'un espace              | Créer un bloc de citation                        |


Ces raccourcis vous aideront à naviguer et à utiliser Notion de manière plus efficace sur Windows. Pour plus de détails, vous pouvez consulter le [Centre d'aide de Notion](https://www.notion.so/help/keyboard-shortcuts?slug=keyboard-shortcuts).
